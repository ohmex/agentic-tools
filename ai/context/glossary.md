# Glossary

> Add a term here whenever a new domain concept is introduced. Keep definitions short and unambiguous.

| Term | Definition |
|---|---|
| Tenant | A customer organization using the platform. All tenant-scoped data carries a `tenant_id`. |
| RLS | Row-Level Security — PostgreSQL feature enforcing tenant isolation at the database layer. |
| ADR | Architecture Decision Record — a short document capturing a significant design decision and its rationale. |
| RFC | Request for Comments — a larger design proposal circulated for cross-team feedback before implementation. |
| OIDC | OpenID Connect — the identity protocol used for authentication. |
| DoD | Definition of Done — the checklist a change must satisfy before being considered complete. |
